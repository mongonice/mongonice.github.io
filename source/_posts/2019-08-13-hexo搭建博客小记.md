## 搭建环境

### 安装hexo

```bash

> npm install hexo-cli -g
```

### 初始化

```bash
> hexo init
```

### 安装hexo部署插件

```bash
> npm install hexo-deployer-git
```

### 安装依赖

```bash
> npm install
```
---

## 写博文

### 在根目录下，创建一篇文章

```bash
> hexo new post "2019-08-13 hexo博客搭建小记"
```
### 先清除再生成静态网页并部署到线上github上

```bash
> hexo clean && hexo g && hexo d
```
